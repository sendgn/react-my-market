## MyMarket react app
A piece of online store (home and product pages)

## Build Setup:

```bash
# Download repository:
git clone https://github.com/sendgn/react-my-market.git
# Go to the app:
cd react-my-market
# Install dependencies:
npm install
# Server with hot reload at http://localhost:3000/
npm start
```
